# Ebook – La Discipline du Succès

Bienvenue dans ce dépôt. Vous trouverez ici :

- `ebook/` — L'ebook complet au format Markdown (prêt à convertir en PDF)
- `carousels/` — Templates de carousels pour TikTok et Instagram *(à venir)*

## Comment convertir l'ebook en PDF

Utilisez [Pandoc](https://pandoc.org/) :

```bash
pandoc ebook/discipline_mindset.md -o ebook/discipline_mindset.pdf
```

Ou copiez le contenu Markdown dans [Notion](https://www.notion.so), [Canva](https://www.canva.com) ou [Visme](https://www.visme.co) pour le mettre en page visuellement avant de l'exporter.
